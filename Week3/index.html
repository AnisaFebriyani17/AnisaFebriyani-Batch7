  1. Apa itu JavaScript?

    2. JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web menjadi lebih interaktif.

  2. Mengapa menggunakan JavaScript?
    • Interactivity (Interaktif Langsung)

      • JavaScript bisa langsung digunakan di browser tanpa harus menginstal apapun. Jadi saat mengklik tombol atau mengisi form, responnya bisa langsung muncul tanpa harus reload halaman.

    • Fleksibel

      • JavaScript bisa bikin tampilan jadi berubah-ubah tanpa reload, membuat animasi biar halaman tidak kaku.

    • Ekosistem Besar

      • Banyak perusahaan yang memakai, tutorial dan dokumentasi mudah ditemukan.

    • Cross-Platform

      • Bisa digunakan selain membuat website, juga untuk membuat aplikasi mobile dan desktop.

  3. Konsep kunci JavaScript
    • Declaration

      • Proses membuat variabel dengan var, let, atau const.

    • Redeclaration (Deklarasi Ulang)

      • Khusus var bisa deklarasi ulang variabel yang sama (tidak disarankan). Tapi let dan const tidak bisa.

    • Reassignment (Mengganti Nilai)

      • let dan var bisa diubah nilainya, const tidak bisa.

    • Hoisting

      • JavaScript akan "mengangkat" deklarasi ke atas. Tapi hanya deklarasinya, bukan nilainya.

    • Global Scope

      • Variabel yang dibuat di luar fungsi bisa diakses dari mana saja di file itu.

    • Function Scope

      • Variabel yang dibuat di dalam fungsi hanya bisa diakses di dalam fungsi itu saja.

    • Block Scope

      • let dan const hanya berlaku di dalam blok {} tempat dia dideklarasikan.
